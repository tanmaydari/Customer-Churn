## 2. Short Description / Purpose
Customer churn is a major challenge for subscription-based businesses.  
This project analyzes customer churn data using **Power BI** to identify churn patterns, high-risk customer segments, and key factors affecting customer retention.

The dashboard presents insights into churn rate, customer tenure, contract types, monthly charges, and retention trends through interactive visualizations.
